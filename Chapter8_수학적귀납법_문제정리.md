## Part3. 수학적 귀납법 (Q19~Q21)

### Thema8. 수학적 귀납법

**Q19.** 모든 자연수 n에 대하여 다음 등식이 성립함을 수학적 귀납법으로 증명하시오.

1/(1×2) + 1/(2×3) + 1/(3×4) + ... + 1/(n(n+1)) = n/(n+1)

**유사문제:**
1. 1 + 2 + 3 + ... + n = n(n+1)/2 을 수학적 귀납법으로 증명하시오.
2. 1² + 2² + 3² + ... + n² = n(n+1)(2n+1)/6 을 수학적 귀납법으로 증명하시오.
3. 2ⁿ > n (단, n ≥ 1)을 수학적 귀납법으로 증명하시오.

### Thema9. 부등식 증명

**Q20.** n ≥ 2인 모든 자연수 n에 대하여 부등식

1 + 1/2 + 1/3 + 1/4 + ... + 1/n > 2n/(n+1)

이 성립함을 수학적 귀납법으로 증명하시오.

**유사문제:**
1. n ≥ 3인 모든 자연수 n에 대하여 부등식 1 + 1/2 + 1/3 + ... + 1/n > 3n/(n+2)이 성립함을 증명하시오.
2. n ≥ 2인 모든 자연수 n에 대하여 부등식 n² > 2n이 성립함을 증명하시오.
3. n ≥ 4인 모든 자연수 n에 대하여 부등식 2ⁿ > n²이 성립함을 증명하시오.

### Thema10. 빈칸 채우기 유형

**Q21.** {aₙ}은 a₁ = 10이고, aₙ₊₁ = Σ(k=1 to n) 2ⁿ⁻ᵏaₖ (n ≥ 1)을 만족시킨다.

다음은 일반항 aₙ을 구하는 과정이다.

【풀이 과정】
주어진 식으로부터 a₂ = (가) 이다.

자연수 n에 대하여
aₙ₊₂ = Σ(k=1 to n+1) 2ⁿ⁺¹⁻ᵏaₖ = Σ(k=1 to n) 2ⁿ⁺¹⁻ᵏaₖ + aₙ₊₁

= (나) Σ(k=1 to n) 2ⁿ⁻ᵏaₖ + aₙ₊₁ = (다) aₙ₊₁이다.

따라서 a₁ = 10이고, n ≥ 2일 때, aₙ = ((다))ⁿ⁻²이다.

위의 (가), (나), (다)에 알맞은 수를 각각 p, q, r라 할 때, p + q + r의 값을 구하시오.

**유사문제:**
1. {aₙ}은 a₁ = 5이고, aₙ₊₁ = Σ(k=1 to n) 3ⁿ⁻ᵏaₖ (n ≥ 1)을 만족시킨다. 일반항 aₙ을 구하고, a₅의 값을 구하시오.
2. {aₙ}은 a₁ = 8이고, aₙ₊₁ = Σ(k=1 to n) 2ⁿ⁻ᵏ⁺¹aₖ (n ≥ 1)을 만족시킨다. a₂와 a₃을 구하시오.
3. {aₙ}은 a₁ = 6이고, aₙ₊₂ = 4aₙ₊₁ (n ≥ 1)을 만족시킨다. 일반항 aₙ을 구하시오.
